# 企业资产管理系统 java480

## 项目概述

企业资产管理系统是基于Springboot+Vue的前后端分离项目，采用MVC三层体系结构，利用Spring、SpringBoot、Mybatis、Vue和Maven技术栈实现。本项目以MySQL为数据存储平台，打造了一套界面简洁、操作简单的B/S结构Web系统。

## 技术栈

- 前端：Vue
- 后端：Springboot
- 数据库：Mybatis
- 数据存储：MySQL
- 项目构建：Maven

## 功能模块

- 资产信息管理
- 资产分类管理
- 资产购置与报废
- 用户权限管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 前端：现代浏览器（推荐使用Chrome、Firefox）
- 后端：Java 1.8及以上版本

## 部署步骤

1. 准备Java运行环境
2. 部署MySQL数据库
3. 导入项目SQL文件
4. 部署后端服务
5. 部署前端项目

## 目录结构

```
前端项目目录
|-- public
|-- src
    |-- assets
    |-- components
    |-- views
后端项目目录
|-- src
    |-- main
        |-- java
            |-- com
                |-- 项目包结构
        |-- resources
|-- pom.xml
```

## 核心亮点

- 前后端分离，易于维护和扩展
- 基于MVC设计模式，结构清晰
- 采用Mybatis，灵活处理数据库操作
- Vue框架实现界面响应式和组件化
- Maven项目构建，便于依赖管理
